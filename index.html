About Me

Hello! I'm Kelly, a software developer based in Pittsburgh, PA.

I'm an aspiring software developer with a background in engineering and data analysis. I'm currently working on a certificate in full stack development from Rutgers University, with newly developed skills in JavaScript and the MERN Stack to add to my existing knowledge of SQL Server and Python.

A lifelong learner, I taught myself SQL to streamline data-gathering tasks for my engineering team. I contributed to building an energy efficiency recommendations engine in Python. My communication and collaboration skills are sharply honed from ten years working as an engineer and project manager.

As a future full-stack or back-end developer, I’m excited to bring data-driven insights to end-users for a fast-paced and innovative company.

When I'm not coding, I can usually be found playing my bass guitar, running, or hanging out with my wife Emily and our dogs.

Portfolio

"Flavor Saver" - The Meal Planning App

Worked with a small group to create a meal planner app, which allows the user to choose meals from a large library and view their weekly plan. My contribution was to design the database schema and models using Sequelize ORM and seed the database through a third-party API.

Employee Tracker CLI

A simple Content Management System (CMS) for small businesses built using Node.js, Inquirer, and MySQL. From this command line interface, the user can view departments, roles, and employees, add departments, roles, and employees, or update a user's role.

"Full Stack Bet" - The Soccer Betting Info Hub

Worked with a small group to create an online hub for European Football information, including upcoming games and team stats. This was a great first experience collaborating with other developers and using GitHub for pull requests, code reviews, and resolving conflicts. My contribution was to use JavaScript’s Fetch API and a third-party API to show the user stats on upcoming matches.

Weather Dashboard

A simple weather dashboard that allows a user to look up current and future weather conditions based on city name. The user's last ten searches will persist in local storage and be shown on the page. The app is built with HTML and CSS, using Bootstrap and Font Awesome for layout and icons. The app is powered by JavaScript, jQuery, and the Open Weather Map APIs.

Password Generator

Best practices for web security tell us to use passwords that are "hard to guess" and to use a different password for each page or app we use. This app will generate a random password for the user, given criteria such as which character types to use and length of desired password The app is built with HTML, CSS, and vanilla JavaScript.
